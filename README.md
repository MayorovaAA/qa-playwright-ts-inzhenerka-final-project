Привет, ревьюер :)

В проекте используется yarn 1 и Node 22

Я позволила себе поэкспериментировать на основании базового домашнего задания, поэтому вот несколько пояснений экстра фич:
- В проекте реализован .env файл. Для локального прогона тестов, пожалуйста, заполните свой локальный env. файл по образцу .env.example.
- Для прогона на CI используются Github Secrets.
- Реализована фикстура, расширяющая тесты авторизованной страницей для того, чтобы избежать логина через UI в каждом тесте.

Это база:
- Реализованы требовавшиеся сценарии и еще несколько.
- Подключен Allure
- Тесты выполняются через GitHub Actions
- Реализован паттерн PageObject
